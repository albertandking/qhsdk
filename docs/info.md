# 资讯

## 实时资讯

### 接口名称

live_news

### 接口描述

实时资讯接口

### 请求参数

|参数名|说明|举例|
|:-----  |:-----|-----     |
|time |资讯最早时间，10位时间戳格式   |1612501467|
|count |返回数据条数，默认100条，最大100条   |50|

### 返回参数

|参数名|类型|说明|
|:-----  |:-----|-----                           |
|content |string   |资讯内容  |
|post_time |datetime   |发表时间，带毫秒格式  |
|cate |string   |资讯分类，目前支持“商品资讯”和“实时资讯”两种  |

### 示例代码

```python
from qhsdk import pro_api
pro = pro_api(token="在此处输入您的token，可以通过联系管理员获取")
live_news_df = pro.live_news(time='1612501467',count="50")
print(live_news_df)
```
